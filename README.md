# Personal Voice Assistant

A Python-based personal voice assistant that can:
- Create and read notes
- Manage a to-do list
- Tell the time and date
- Explain scientific concepts
- Share periodic table facts
- Give motivational quotes

## Setup

1. Create a virtual environment with Python 3.10
2. Install dependencies: `pip install -r requirements.txt`
3. Install PyAudio: download wheel from https://pypi.org/project/PyAudio/#files
4. Download NLTK data: `python -c "import nltk; nltk.download('punkt')"`
5. Run: `python main.py`
